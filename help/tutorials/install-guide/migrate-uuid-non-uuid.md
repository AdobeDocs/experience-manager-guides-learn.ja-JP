# 非 UUID から UUID へのコンテンツの移行 {#id226TI0U20XA}


UUID 以外のコンテンツを UUID に移行できます。

>[!IMPORTANT]
>
> コンテンツを UUID サーバーに移行する前に、AEMガイドバージョン 4.0 または 4.1 がインストールされた非 UUID サーバーがあることを確認してください。
