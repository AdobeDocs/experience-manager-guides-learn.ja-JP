# 酸素を使用する {#id221BH0V0UE8}

Oxygen XML Author は、使いやすいオーサリング機能を備え、使いやすいインターフェイスを提供します。

同じソースを使用して、PDF、ePub、HTML、その他多くの形式の出力を生成できます。

*この機能を環境で有効にするには、カスタマーサクセスチームにお問い合わせください。 これは、標準のサポートの一部としては有効になっていません。*
