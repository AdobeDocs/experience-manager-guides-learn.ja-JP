# マップダッシュボードの概要

次に、マップダッシュボードの主な機能の概要を説明します。

>[!VIDEO](https://video.tv.adobe.com/v/339040?quality=12&learn=on)

## マップダッシュボードでマップを開く

1. リポジトリビューで、マップ上の省略記号アイコンを選択してオプションメニューを開き、「マップダッシュボードを開く」を選択します。
   ![images/ellipsis-map-dashboard.png](images/ellipsis-map-dashboard.png)

   マップダッシュボードが別のタブで開きます。

## マップダッシュボードのコンポーネント

マップダッシュボードには、出力プリセット、出力結果、使用したトピック、ベースラインなど、複数のタブが含まれます。

### 出力プリセット

「出力プリセット」タブには、様々な種類の出力（AEM サイト、PDF、HTML 5、ePub、カスタム）のデフォルトプリセットが表示されます。

![images/output-presets.png](images/output-presets.png)

出力プリセットを選択して、変換名、宛先パス、ベースライン、適用された条件など、設定の詳細を表示できます。

### 出力

「出力」タブには、以前に生成された出力と現在生成されている出力がすべて表示されます。

![images/generated-outputs.png](images/generated-outputs.png)

「生成設定」列の下の緑の円は、出力が正常に生成されたことを示します。 この列のテキストはアクティブなハイパーリンクとして機能し、それらを選択して生成された出力を開くことができます。 タイプ列の下のエントリは、出力タイプを示します。
出力を生成したユーザーの名前、生成日時、生成にかかった時間など、その他の出力生成情報もここに表示されます。 生成中にエラーが発生した場合は、「生成時」列で生成日時を選択して、エラーログを開いて確認できます。

### トピック

「トピック」タブには、マップ内のすべてのトピックのリストが表示されます。

![images/topics.png](images/topics.png)

トピックのチェックボックスを選択すると、追加のアクションを実行できます。 編集、再生成、タグの表示、適用、非表示を行うことができます。

### 条件プリセット

「条件プリセット」タブには、特定の条件付きコンテンツを含めるか除外するかを指定する設定が表示されます。

![images/condition-presets.png](images/condition-presets.png)

ここで、「ライターのみ」エディションのチェックボックスを選択すると、「designers」ラベルを持つ「audience」属性を持つすべてのコンテンツが除外され、「writers」ラベルを持つすべてのコンテンツが含まれる出力が生成されます。

### ベースライン

「ベースライン」タブでは、ベースラインを表示できます。

![images/baselines.png](images/baselines.png)

ベースラインは、ある時点でスナップショットとして機能し、トピックとアセットのバージョンを作成して公開できます。 例えば、特定の日時のコンテンツをキャプチャするベースラインでは、1つのトピックのバージョン 1.3と別のトピックのバージョン 1.0を、それぞれのバージョンに基づいて使用できます。
ベースラインが指定されていない場合、出力はすべてのコンテンツの最新バージョンで生成されます。

### レポート

「レポート」タブには、使用中のトピックの総数、これらのトピック内の欠落している要素、ドキュメントの状態など、トピック情報の概要が表示されます。

![images/reports.png](images/reports.png)

トピックに要素がない場合は、行の右端の矢印を選択してエントリを展開し、エラーに関する詳細を表示できます。
