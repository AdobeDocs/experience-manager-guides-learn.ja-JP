# AEM ユーザーインターフェイスについて

Experience Manager用XML エディターでのアセット、ファイル、フォルダーの管理について説明します。

>[!VIDEO](https://video.tv.adobe.com/v/336659?quality=12&learn=on)

## XML エディターへのアクセス

1. ログイン画面で、ユーザー名とパスワードを入力し、**[!UICONTROL ログイン]**&#x200B;を選択します。
1. AEM ナビゲーションページから「**[!UICONTROL XML Editor]**」を選択します。

## XML エディタービュー

サイドバーから、組織のニーズに最適なXML エディターの複数のビューから選択できます。 これには、デフォルトのビューと[!UICONTROL  リポジトリビュー]が含まれます。

既定のビューには、[!UICONTROL お気に入り]が表示されます。 必要に応じてショートカットを使用して、さらにカスタマイズすることができます。 逆に、[!UICONTROL  リポジトリビュー]では、従来のフォルダー構造が表示されます。

### デフォルトのビューから[!UICONTROL  リポジトリビュー]に切り替える

1. 左側のパネルで、**[!UICONTROL リポジトリビュー]**&#x200B;を選択します。

   ![ リポジトリアイコン ](images/common/repository-icon.png)

   [!UICONTROL  リポジトリビュー]が表示されます。

## AssetsのUI

[!UICONTROL Assets] インターフェイスでは、以下を表示できます

### Assets UIでのコンテンツの表示

コンテンツに対して追加のアクションを実行できます。 これらのオプションの1つは、Assets UIでファイルを表示することです。

1. リポジトリ内のフォルダーまたはトピックにカーソルを合わせ、表示される省略記号アイコンを選択します。

   ![省略記号アイコン ](images/lesson-2/options-menu-with-markings.png)

   オプションメニューが表示されます。

1. メニューから、「**Assets UIで表示」を選択します。**

   ![Assets UI](images/lesson-2/assets-ui.png)での表示


### ビューの選択

Assets UIでは、組織のニーズに最適な複数のビューから選択できます。

1. 右上隅の「**スイッチャーを表示**」アイコンを選択します。

   ![切り替えアイコンを表示](images/lesson-2/view-switcher.png)

   ドロップダウンメニューが表示されます。

1. 使用するビューを選択します。

### Assets UI ビュー

| 名前 | 説明 |
| --- | --- |
| カード表示 | 各アセットをアイコンとして表示 |
| 列表示 | 圧縮された拡張可能なフォルダー構造でアセットを表示します |
| リスト表示 | リスト内のアセットとその詳細を表示します |

## XML エディターに戻る

Assets UIの任意のビューからXML エディターに戻ることができます。

### リストおよび列表示から戻る

1. リストから編集するトピックを選択します。
Assets UIにトピックが表示されます。
1. 上部のツールバーから「**編集**」を選択します。
XML エディターに戻されます。

### カード表示から戻る

1. [!UICONTROL Assets] UIでトピックの鉛筆アイコンを選択します。

   ![鉛筆アイコン ](images/lesson-2/return-card-view.png)

   XML エディターに戻されます。
